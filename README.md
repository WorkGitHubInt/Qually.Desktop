# Qually.Desktop
Основная работа была проведена над Qually.Desktop. Это репозиторий, содержащий в себе 3 десктоп приложения на WPF. Приложения распространяются по месячной подписке на сайте https://botqually.ru , также в группе https://vk.com/botqually . Приложения в этом репозитории (Qually.Desktop) - это боты для браузерной игры lowadi.com. Основной принцип работы это парсинг HTML и JSON и отправка запросов на выполнение действий. Через них осуществляется вход в игровые аккаунты и выполнение рутинных действий в игре. Все 3 приложения выполнены приблизительно одинаково (по признаку используемых средств при разработке), только выполняют разные задачи в рамках игры. По значимому коду AccountHolder - самое маленькое приложение, а QuallyFlash - самое большое.
# Qually.Web
Репозиторий содержит всё, что касается сайта botqually.ru и месячной подписки.
Основные фреймворки: ASP.NET Core, EntityFramework Core.
Qually - это основной сайт для пользователей, на нём расположена вся информация по распространяемым приложения и API для работы с месячной подпиской.
AdmilQuallyMVC - это администраторский сайт для контроля БД пользователей.
QuallyLauncher - это десктоп клиент, написанный на Javascript с использованием Electron'a. Оттуда обычно ведется управление распространяемыми программами.
