# Qually.Desktop
На этом ГХ аккаунте собраны практически все написанные приложения за 1.5 года.
Основная работа была произведенная над Qually.Desktop. Это репозиторий содержащий в себе 3 десктоп приложения на WPF. Приложения распространяются по месячной подписке на сайте https://botqually.ru , также в группе https://vk.com/botqually . Приложения в этом репозитории (Qually.Desktop) - это боты для браузерной игры lowadi.com . Через них осуществляется вход в игровые аккаунты и выполнения рутинных действий в игре. Все 3 приложения выполнены +- одинаково, только выполняют разные задачи в рамках игры.
# Qually.Web
Репозиторий содержит всё, что касается сайта botqually.ru и месячной подписки.
Qually - это основной сайт для пользователей, на нём вся информация по распространяемым приложения и API для работы с месячной подпиской.
AdmilQuallyMVC - это администраторский сайт для контроля БД пользователей.
QuallyLauncher - это десктоп клиент написанный на Javascript с использованием Electron'a оттуда обычно ведется управление распространяемыми программами.
